# チケット一覧表示コマンド

`./Documents/tickets/` 内のチケット一覧を表示し、現在の進捗状況を把握します。

## 実行フロー

### 1. チケットファイルの取得

```bash
# チケットディレクトリの確認
ls -la ./Documents/tickets/
```

### 2. 各チケットの情報抽出

各チケットファイルから以下の情報を抽出：
- チケットID
- タイトル
- ステータス
- 優先度
- 担当
- 作成日

### 3. 一覧表示

#### 3.1 サマリー表示

```
═══════════════════════════════════════════════════
📋 チケット一覧
═══════════════════════════════════════════════════

総数: XX件
├─ Open: X件
├─ In Progress: X件
├─ Review: X件
├─ Closed: X件
└─ Draft: X件

───────────────────────────────────────────────────
```

#### 3.2 アクティブチケット一覧

ステータスが Draft / Open / In Progress / Review のチケットを表示：

```
📌 アクティブチケット
───────────────────────────────────────────────────
| ID | タイトル | ステータス | 優先度 | 担当 |
|----|----------|-----------|--------|------|
| ticket-dev-001 | [タイトル] | In Progress | 🟡 高 | Claude |
| ticket-fix-002 | [タイトル] | Review | 🔴 緊急 | Claude |
| ticket-dev-003 | [タイトル] | Open | 🟢 通常 | - |
───────────────────────────────────────────────────
```

#### 3.3 最近クローズしたチケット

直近5件のクローズ済みチケット：

```
✅ 最近クローズしたチケット（直近5件）
───────────────────────────────────────────────────
| ID | タイトル | クローズ日 |
|----|----------|-----------|
| ticket-dev-001 | [タイトル] | 2025-01-20 |
───────────────────────────────────────────────────
```

### 4. フィルタリングオプション

引数による絞り込み（$ARGUMENTS）：

| 引数 | 説明 |
|------|------|
| `--open` | Open ステータスのみ |
| `--progress` | In Progress ステータスのみ |
| `--review` | Review ステータスのみ |
| `--closed` | Closed ステータスのみ |
| `--dev` | 開発チケットのみ |
| `--fix` | 修正チケットのみ |
| `--high` | 高優先度以上のみ |

### 5. 詳細表示オプション

特定チケットの詳細を表示（$ARGUMENTS にチケットファイル名を指定）：

```
═══════════════════════════════════════════════════

═══════════════════════════════════════════════════

優先度: 🟡 高
作成日: 2025-01-15
担当: Claude Code

───────────────────────────────────────────────────
完了条件
───────────────────────────────────────────────────
[✓] 機能要件1
[ ] 機能要件2
[ ] テスト基準

進捗: 33% (1/3)

───────────────────────────────────────────────────
最近の更新
───────────────────────────────────────────────────
- 2025-01-20 15:30: ステータス変更 (Open → In Progress)
- 2025-01-18 10:00: ユーザーレビュー承認

═══════════════════════════════════════════════════
```

## 出力形式

### コンパクト表示（デフォルト）

簡潔な一覧形式で表示します。

### 詳細表示（--verbose）

各チケットの完了条件の進捗も含めて表示します。

## 使用例

```bash
# 全チケット一覧
/list-tickets

# 進行中のチケットのみ
/list-tickets --progress

# 高優先度の開発チケット
/list-tickets --dev --high

# 特定チケットの詳細
/list-tickets ticket-dev-001.md
```

## 注意事項

- チケットがない場合は「チケットがありません」と表示
- ファイル形式が不正な場合はスキップして警告を表示
- 大量のチケットがある場合は直近20件のみ表示（--all で全件表示）
