プライバシーポリシー

最終更新日：2026年3月5日

SecureZip（以下「本アプリ」）は、個人開発者 murakami-akane が提供する macOS 向けアプリケーションです。本ポリシーは、本アプリがどのような情報を取り扱うかを説明します。

1. 収集する情報

本アプリが収集・保存する情報は以下のみです：

  • Gmail メールアドレス：Gmail 連携時に取得します。お使いのデバイスの Keychain にのみ保存され、外部サーバーには送信されません。
  • 送付履歴：送付先メールアドレス・ファイル名・送信日時をデバイス内の Core Data（ローカルデータベース）に保存します。設定した日数が経過すると自動削除されます。

2. 収集しない情報

本アプリは以下の情報を一切収集しません：

  • アナリティクス・利用統計データ
  • 広告識別子
  • 位置情報
  • ファイルの内容
  • クラッシュレポート（開発者への自動送信なし）

3. Gmail API の利用

本アプリは Gmail API を通じてメール送信を行います。使用する OAuth 2.0 スコープは gmail.send のみであり、メールの読み取り・削除・連絡先へのアクセスは行いません。認証トークンはデバイスの Keychain に保存され、開発者を含む第三者がアクセスすることはできません。

本アプリの Google API の利用は Google API Services User Data Policy に準拠しています。

4. 第三者への情報提供

本アプリは、ユーザーの情報を第三者に販売・提供・共有しません。

5. データの保管場所

すべてのデータはユーザーのデバイス上にのみ保存されます。開発者のサーバーやクラウドサービスにデータが送信されることはありません。

6. データの削除

アプリをアンインストールすることで、Core Data に保存されたすべての履歴データが削除されます。Keychain に保存された認証情報は、アプリ内の「設定 → Gmail 連携解除」から削除できます。

7. お子様のプライバシー

本アプリは 13 歳未満のお子様を対象としておらず、意図的にお子様の情報を収集することはありません。

8. ポリシーの変更

本ポリシーは必要に応じて更新されることがあります。変更がある場合は、このページの「最終更新日」を更新します。

9. お問い合わせ

本ポリシーに関するご質問は、App Store のサポートページよりお問い合わせください。

Privacy Policy

Last updated: March 5, 2026

SecureZip ("the App") is a macOS application developed by murakami-akane. This policy explains what information the App handles.

1. Information We Collect

The App collects and stores only the following information:

  • Gmail email address: Obtained when connecting to Gmail. Stored only in your device's Keychain and never transmitted to external servers.
  • Send history: Recipient email address, file name, and send date/time are stored in Core Data (local database) on your device. This data is automatically deleted after the number of days you configure.

2. Information We Do Not Collect

The App does not collect any of the following:

  • Analytics or usage statistics
  • Advertising identifiers
  • Location data
  • File contents
  • Crash reports (no automatic reporting to the developer)

3. Use of Gmail API

The App uses the Gmail API solely to send emails. The only OAuth 2.0 scope used is gmail.send. The App does not read, delete, or access your emails or contacts. Authentication tokens are stored in your device's Keychain and cannot be accessed by the developer or any third party.

The App's use of Google APIs complies with the Google API Services User Data Policy.

4. Sharing of Information

The App does not sell, provide, or share your information with any third party.

5. Data Storage

All data is stored solely on your device. No data is transmitted to the developer's servers or any cloud service.

6. Data Deletion

Uninstalling the App will delete all history data stored in Core Data. Authentication credentials stored in Keychain can be removed via Settings → Disconnect Gmail within the App.

7. Children's Privacy

The App is not directed at children under the age of 13, and we do not knowingly collect information from children.

8. Changes to This Policy

This policy may be updated from time to time. When changes are made, the "Last updated" date on this page will be revised.

9. Contact

For questions about this policy, please contact us via the support page on the App Store.
